## Usage

To run the SQL commands:
1. Connect to MySQL: `sudo mysql`
2. Create database: `CREATE DATABASE dswork;`
3. Use database: `USE dswork;`
4. Run commands: `SOURCE ~/DBS/Prac01/Prac01Part1Commands;`
